8 Typographic Systems

Here is my design of the typography poster in each typographic system.

Axial

Axial is like a page split into two pages in an imaginary line. In my design, I used black and white, the two contrasting colours to create a line and put this text on both sides.

Radial

Radial typography is simple. As you can see, it is like radiation. In my design, I created a circle and put it on the left side and put the text around the circle.

Dilatational

In my dilatational typography, I use some circles to wrap the text. I put the text layer by layer. And you can see the text is like surrounds an invisible circle.

Random

As the name of it, the text is randomly added to the page. That is why you can see some of the text is overlapping and bold.

Grid

Grid typography is using by horizontal and vertical lines to make some layouts. In my design, I used the grid function to help me adjust the size of the text and shape.

Modular

Modular typography is designed with different modules by some shape or line.I used different sizes of circles to create a different module but the circles are connected together because I want to show the connection of different modules.

Transitonal

Transitional typography is like something transiting from one point to another point. I started with the pen tool to draw some wavy lines and put the text on the lines.

Bilateral

The text and shape in Bilateral typography were fitted in the middle section. I feel like it is boring if I only add text. That is the way I also put some shapes between the text.
